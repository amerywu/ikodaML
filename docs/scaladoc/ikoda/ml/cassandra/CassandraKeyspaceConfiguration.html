The configuration parameters for a keyspace. Loaded from keyspaces.conf

flusho

true if this keyspace will be flushed

trimo

true if low frequency columns should be removed during fush

supplemento

true if supplementary dense data expected

hdfso

true if directories are on Hadoop

truncateoverwriteo

true if keyspace data is to be overwritten with each batch

archiveo

true if incoming files shou,d be archived

flushthresholdo

staging keyspace rowcount that triggers flush

uidinsparseo

true if sparse data shares a UID with supplementary dense data

diro

the staging directory for sparse data

persisteddiro

the destination for files already persisted to Cassandra

errordiro

the destination for files that failed during persistence process

supplementdiro

the staging directory for supplementary dense data

supplementpersisteddiro

the destination for supplementary files that failed during persistence process

supplementerrordiro

the destination for supplementary files that failed during persistence process

supplementsuffixo

the filename suffix that identifies a supplementary data file

uuido

the uuid column name

permittedProportionOfDuplicateso

the permitted proportion of duplicate rows that can be persisted.
